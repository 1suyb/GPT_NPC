# GPT_NPC
**Unity 에서 사용자 음성을 입력으로 받아 출력으로 아바타의 응답과 행동을 생성하는 프로젝트**

### 언어 및 통신방법
Front : Unity (C#)
Back : Python
통신 : Socket

### 사용 모델
Whisper-base, finetuning

### API
OpenAI API - GPT3.5 turbo
